Algorithms Cheat Sheet

  • Breadth-First Search traverses tree or graph data structures by visiting closest nodes first
  • Depth-First Search traverses tree or graph data structures by exploring as far as possible before backtracking
  • Binary Search finds the position of an element within a sorted array
  • Merge Sort sorts an array
  • Quick Sort sorts an array
Algorithm Runtime Notes
Breadth-First Search O(V + E) (worst case) Solves shortest paths with unit weight edges
Depth-First Search O(V + E) (worst case) Solves a multitude of problems including paths, cycles, sorts
Binary Search O(log N) Only works on sorted arrays
Merge Sort O(N log N) Stable (retains order of identical items)
always same runtime
Quick Sort O(N log N) average case
O(n^2) worst case		 Unstable (does not retain order of identical items)
fastest in practice
